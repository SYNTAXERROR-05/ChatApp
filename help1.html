Basic Features

Privacy & Security

End-to-End Encryption

All messages are secured with end-to-end encryption. Only you and the recipient can read your conversations.

Privacy Settings

Control who can see your profile, status, and contact you.

  1. Go to Settings > Privacy
  2. Adjust who can see your profile picture
  3. Control last seen status visibility
  4. Manage read receipts

Blocking Users

If you need to block someone:

  1. Open the user's chat
  2. Tap their name at the top
  3. Scroll down and select "Block User"
  4. Confirm your choice

Data Deletion

Learn how to delete your data and account if needed:

  1. Go to Settings > Account
  2. Select "Delete My Account"
  3. Follow the confirmation steps

Troubleshooting

If you're having trouble connecting:

  1. Check your internet connection
  2. Ensure you're not in airplane mode
  3. Try switching between Wi-Fi and mobile data
  4. Restart the app
  5. Update to the latest version

If your messages aren't being delivered:

  1. Check for a stable internet connection
  2. Verify the recipient hasn't blocked you
  3. Look for a single checkmark (sent) vs double checkmark (delivered)
  4. Try sending the message again after a few minutes

If translations aren't working correctly:

  1. Verify language settings are correct
  2. Try using simpler sentences with fewer idioms
  3. Report inaccurate translations via the feedback button
  4. Check for app updates that might improve translation quality

If the app is unstable:

  1. Close and restart the app
  2. Update to the latest version
  3. Clear app cache (Settings > Apps > Transla Talk > Clear Cache)
  4. Ensure your device has enough storage space
  5. As a last resort, reinstall the app

Accessibility

  • TalkBack (Android)
  • All elements have proper labels and descriptions

Contrast Options

Improve visibility with high contrast mode:

  1. Go to Settings > Accessibility > High Contrast
  2. Toggle the switch to enable/disable

Hearing Assistance

Features for hearing-impaired users:

  • Visual notifications for calls and messages
  • Vibration patterns customization
  • Live captions during voice/video calls
